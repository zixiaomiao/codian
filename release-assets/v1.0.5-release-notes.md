# Codian v1.0.5

## 中文说明

这个版本重构了分类系统，修复了关键词误判和数据静默丢失的问题，同时增加了 Hermes Agent 的支持。

### 核心修复

- **分类系统重构**：将散落在 8 处的分类定义统一到 `CATEGORIES` 注册表 dict。`classify_log`、`categories_for_query`、`category_purpose` 自动派生，扩展新分类改一处即可，不再漏改丢数据。
- **修复关键词误判**：移除了 `classify_log()` 中过于宽泛的关键词（如"改为"、"不要"、"验证"），日志分类更准确。修复日志不再因"改为"一词被误判到 Decision。
- **新增 orphan 交叉检查**：`generate_memory_categories()` 写入前校验 buckets 中所有分类是否有对应文件路径，无路径映射时输出警告到 stderr，不再静默吞数据。
- **`.gitignore` 移至仓库根目录**：避免 `.DS_Store`、`.env` 等文件被意外提交。
- **安装脚本支持多 Agent**：`install.sh` / `install.ps1` 新增 `hermes` 安装模式，安装到 `~/.hermes/skills/codian/`。
- **新增 Hermes skill 目录**：`hermes/SKILL.md` + `scripts/obsidian_memory.py`，供 Hermes Agent 直接使用。

### 文件变动

| 文件 | 变动 |
|------|------|
| `.gitignore` | 移至仓库根目录 |
| `Codian/.codex-plugin/plugin.json` | 版本更新至 1.0.5 |
| `Codian/scripts/obsidian_memory.py` | 分类系统重构 + 关键词修复 + orphan 检查 |
| `hermes/SKILL.md` | 新增 |
| `hermes/scripts/obsidian_memory.py` | 新增（与 `Codian/scripts/` 同步） |
| `install.sh` | 支持 `codex` 和 `hermes` 双目标安装 |
| `install.ps1` | 同上 |

### 安装

Codex:

```bash
curl -fsSL https://raw.githubusercontent.com/zixiaomiao/codian/main/install.sh | bash
```

Hermes:

```bash
curl -fsSL https://raw.githubusercontent.com/zixiaomiao/codian/main/install.sh | bash -s hermes
```

或直接从 GitHub 下载 `hermes/` 目录复制到 `~/.hermes/skills/codian/`。
