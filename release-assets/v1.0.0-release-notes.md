# Codian v1.0.0

This release promotes the plugin to Codian v1.0.0 and syncs the expanded Codex skill instructions into the GitHub plugin repository.

## Highlights

- Renames the Codex plugin identity to `codian`.
- Updates the plugin manifest version to `1.0.0`.
- Moves the bundled skill to `skills/codian/SKILL.md`.
- Expands the skill instructions with the full Obsidian memory workflow:
  - compact memory reads
  - keyword-matched historical log retrieval
  - safe append workflow
  - project summary generation
  - categorized memory generation
  - vault discovery rules
- Keeps the memory script focused on the structured Obsidian vault layout:
  - `README.md`
  - `AGENTS.md`
  - `10-Context-上下文/`
  - `20-Memory-记忆/`
  - `30-Logs-日志/`
  - `40-Workflows-工作流/`
  - `90-Archive-归档/`
